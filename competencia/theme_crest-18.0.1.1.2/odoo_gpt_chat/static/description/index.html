How To Use This App?

Step 1: Sign Up

Create an account on https://www.whisperchat.ai and access the create the chatbot.

Step 2: Link or Upload

Add your website link or upload relevant documents.

Step 3: Train the BOT

Our AI will analyze the data and train your personalized chatbot.

- Deploy

Add the chat widget to your Odoo website by adding the chatbot ID in configuration.

Open odoo integration setting

- Go Live

Your AI-powered chatbot is now ready to engage with visitors!

Note :

  • If you use multiple databases, add this line in your odoo config file to avoid setting a specific database name.
  • server_wide_modules = base,web,odoo_gpt_chat
  • After adding this, the odoo_gpt_chat module will be loaded globally for all databases.Then, just add your database name inside the whisperchat website settings.
  • Important:
    If you skip this step, the module may not work and whisperchat might fail to load in odoo.

Step 4: Odoo ERP setup

In the chatbot settings, add your Odoo database URL to connect the chatbot with your system.

- Bot Added to Odoo Database

Once the URL is added, the chatbot will be automatically integrated into your Odoo database.

- Public User Created

A Public User is automatically created in the database to handle chatbot interactions.

- Assign CRM and Messages Access to User

After the user is created. Go to user > chatbot access and assign access to the CRM & Messages modules.

- Lead Creation via Chatbot

Once access is granted, the user can send their information through the chatbot - a new lead will
be automatically created in the CRM Leads menu.

Step 5: Cal.com Scheduler

Integrate Cal.com meeting scheduler for appointments.
