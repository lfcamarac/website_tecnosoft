========================
Theme Zenith
========================

The Ultimate High-Performance eCommerce Theme for Odoo 18.

.. image:: static/description/banner.png
   :alt: Theme Zenith Banner
   :width: 100%

Key Features
============

*   **Multi-Style Engine**: Switch between Minimalist, Modern, Futuristic, Epic, and Dynamic styles instantly.
*   **Premium Header Variants**: Glassmorphism, Sticky, Centered, and more.
*   **Dynamic Snippets**:
    *   **Hero Section**: With image hotspots and branding.
    *   **Timeline**: Fully editable history builder.
    *   **Pricing Table**: Interactive plans with "Most Popular" toggle.
    *   **Features Grid**, **Logo Slider**, **Category Tabs**.
*   **Zenith Side-Cart**: Smooth slide-out cart for uninterrupted shopping.
*   **Scroll Reveal**: Cinematic animations on scroll.
*   **Multi-Currency & Multi-Language**: Built-in support for global sales.
*   **Retail Ready**: Includes demo data and professional polish.

Configuration
=============

1.  Go to **Website > Editor**.
2.  Select the **Theme** tab (or Main options) to pick your **Zenith Style**.
3.  Drag and drop snippets from the **Zenith Building Blocks** section.
4.  Visit `/zenith-showcase` to see a demo page.
